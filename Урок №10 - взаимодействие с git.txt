Для добавления файла в первый раз и инициализация репозитория:
cd <адрес папки>   - заходим в нужную папку
git init   - инициализируем папку как локальный репозиторий
git status   - проверяем статус файлов в папке
git add <имя файла>   - добавляем файл в список добавляемых
git commit -m 'сообщение'   - закрепляем файл в репозитории с комментарием
git remote add origin <ссылка на облачный репозиторий>   - устанавливаем связь локального и облачного репозиториев (ДЕЛАЕТСЯ ЛИШЬ РАЗ)
git branch -M main   - тоже устанавливаем связь видимо хз (ДЕЛАЕТСЯ ЛИШЬ РАЗ)
git push   - закидываем закреплённый список файлов на облачный репозиторий
